How it works

Four steps to the apex.

01

Detect Your Wheel

Auto-scans all connected controllers. Known sim hardware is highlighted. Moza, Fanatec, Simagic, and more detected automatically.

02

Find Forza

Auto-detects your Forza Horizon install on Steam and Xbox Game Pass. Browse manually if needed.

03

Map Controls

Press each input when prompted. Covers all 26 bindings: steering, pedals, paddles, face buttons, D-Pad, and more.

04

Generate & Install

One click creates a backup, builds the patched ZIP, and installs it straight to your game's media folder.

Quick Remap

After setup, remap any single binding without re-running the full wizard.

Backup & Restore

Originals backed up to HST-BACKUP before anything is touched. One-click restore.

FFB Tuning

Optional force feedback templates for compatible wheels via the Advanced panel.

No Server, No Port

Fully local — WebView2 renders the UI inside the app. Nothing reaches the network.

Built on

Horizon SimTool by Dxniel02

Horizon Wheel Wizard is built on top of Horizon SimTool by Dxniel02. The core backend — device enumeration, XML/INI patching, zip generation, game file installation, backup/restore, and the original mapping engine — all come from that project. This tool adds a WebView2 wizard UI on top of that foundation.
