Wheel Mapping Wizard
The wizard walks you through pressing each pedal and button to map all 26 inputs including steering, pedals, paddles, buttons, and map/view. It supports multiple USB devices (e.g. MOZA wheelbase + SIMSONN pedals) simultaneously.
Before you start
  • ✓ Wheelbase must be assigned in Device Roles
  • ✓ Connect all devices before launching
  • ✓ Center your steering wheel before starting
  • → Input mapping profile will be generated automatically
